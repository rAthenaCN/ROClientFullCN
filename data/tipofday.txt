→/savechat  :  把对话窗口的内容储存起来
#
→按住ctrl键，再按*键，可把微型地图从半透明变成不透明，或是不透明变成半透明 
#
→按住shift键，再按鼠标右键拖曳，可以把视角的角度抬高，如要恢复原状，按住shift键，再按两次鼠标右键，既可恢复 
#
→按住ctrl键，再按鼠标右键拖曳，可以把视角放大缩小，这时按住ctrl键，再按两次鼠标右键，既可恢复原状，用鼠标滚轮也可以把画面放大缩小
#
→把光标放在敌人身上时，游标会变成刀， 既可攻击
#
→从魔物那里得到的经验值，是与魔物受到的损害力同正比，
   ※一起攻击魔物时，先攻击魔物的人，得到的经验值较多
#
→按住ctrl键攻击魔物时，会变成[锁定攻击状态]，将会持续自动攻击到魔物死亡为止，想要解除自动攻击状态时，将鼠标光标点选其它位置既可
#
→想要捡掉下来的物品，把光标指在物品上再[点选鼠标左键]即可，捡到的物品会放在物品栏里
#
→把光标移到物品身上[点选鼠标右键]时，可以看到物品的情报
#
→玩家脚底下二个横杆是玩家的HP值和SP值，绿色的表示HP值，蓝色的表示SP值，HP值到0时，表示玩家已死亡，为了警惕玩家HP值剩下25%以下时，HP值会改变成显示为红色
#
→EXP显示杆已满的话(100%)，基础等级会上升，基础等级上升时，会获得素质点数（Status Point），玩家可自由加给于想培养的素质
#
→EXP 表示杆下面的JOBEXP显示杆已满的话(100%)， 职业等级会上升，会获得技能点数，玩家可自由加给于想培养的技能
#
→若玩家不动于原地时，HP和SP会稍微恢复
   ※坐着时会恢复速度会加快两倍，而当重量超过50%以上时，将不会恢复HP值与SP值
#
→Weight(重量) : 显示为最大负重量和现在携带的物品重量，玩家无法携带超过最大负重量的物品，物品重量超过最大负重量时，将不能携带任何物品。若携带的物品重量超过最大负重量一半时，将会显示为红色，这时HP和SP不会自动恢复，还有超过最大负重量90%时，玩家将不能攻击和使用技能
#
→Zeny(金币) : 金币是在仙境传说里所使用的[金钱]单位 
   ※玩家所拥有的金币当交易或购买物品时会自动扣减
#
→基础等级上升后，会得到素质点数（Status Point）， 玩家可利用这点数培养素质
从1到10等级会得到3点数 ，每5等级会增加1点数
#
→STR(力量) : 对玩家攻击力有直接的影响 ，力量增加时，最大负重量也将会增加，将可携带更多物品。力量增加将会增加近距离的攻击力
#
→DEX(灵巧) : 对玩家使用武器时有直接的关系.
数值越高，越能发挥武器的攻击力，用弓箭攻击时，因DEX值提高也将会提高攻击力，也会增加命中率，也影响到施放魔法的速度
#
→AGI(敏捷) : 对玩家的行动敏捷度有直接的关系.
增加这数值时，回避率将会增加，可闪躲敌人的攻击。而各种武器类的攻击速度也将会提升.
#
→VIT(体质) : 对玩家的防御力和HP值多寡有直接的关系.
   数值越高，防御力也将会增加，遭受敌人攻击时损失可减低，还有将会增加最大HP值和加快恢复HP的速度.
#
→INT(智力) : 对玩家的魔力（SP）有直接的关系，数值越高，将会增加魔法攻击力（MATK）和魔法防御力（MDEF）
#
→LUK(幸运) : 玩家的幸运对各种状况都有直接的关系，主要是必杀攻击率和完全回避率，都和幸运增加有直接关系
#
→[Print Screen] or [Scroll Lock] : 拍照，照片会在[ScreenShot] 档案内存盘起来
#
→[F10] : 可以调查对话窗口大小
#
→[F11] : 只保留基本情报窗口和对话窗口，关闭其它窗口
#
→[F12] : 可以开启关闭快速使用列  
   ※将经常使用的物品或是技能拖曳于在F1～F9键上，直接按所属快速键就可以使用
#
→[INSERT] :将可坐下或站立  
※将[/坐下]或是[/站立]这两个指令直接输入于对话窗就可以使用。在坐下的状态时HP、SP回复速度比站立时加快两倍
#
→[Alt] + [End] : 可以把角色下方 [HP / SP] 图示开启/关闭
#
→[Alt] + [Home] : 可以把可移动的地区图示开启/关闭
#
→[Alt] + [=] : 切换输入状态，全角转半角。用全角输入的命令不会执行 这时候切换成半角输入即可
#
→[Alt] + [F10] : 可以把聊天窗口开启/关闭
#
→[Alt] + [M] :  让各种表情符号 / 重复的聊天的文字，让他重复的储存。
#
→ [Alt] + [1~9] : [ALT] + [M]可以使用已储存好的内容
#
→ [Ctrl] + [-,=,w]U可以使用 剪刀 / 石头 /布  表情符号
#
→/WHO : 可以看到服务器上， 在线玩家总人数
#
→/WHERE : 可以看到现在所在位置的情报和坐标
#
→/H : 显示对基本的指令说明
#
→/BGM : 可以把背景音乐开启/关闭  
   ※也可以按 [Alt] + [0] 在窗口上执行
#
→/SOUND : 可以把效果音开启/关闭   
   ※也可以按 [Alt] + [0] 在窗口上执行
#
→/EFFECT : 可以把游戏里的各种效果图示开启/关闭   
   ※关闭效果时，可以使游戏执行上更加顺畅
#
→/SKIP :开启/关闭画面同步更新
   ※当画面关闭时，因计算机配备不同会造成游戏速度下降，当游戏画面继续有Lag时建议你把他关掉
#
→/EX【角色名称】:  可以隔绝此玩家所说的话和悄悄话
→/IN【角色名称】:  恢复与此玩家之间的对话
#
→/EXALL : 把所有角色的对话隔绝
→/INALL : 恢复与所有玩家对话
#
→/EX : 显示被隔绝玩家的清单
#
→/ORGANIZE 【队伍名称】: 把输入的名字可以组成组队员
   ※组队后变成组队长时，可以邀请玩家参加组队
#
→把光标放在想邀请的玩家身上， 点选鼠标右键，会显示 ' 邀请参加组队 ' 选项，利用这选项，可组成组队 
#
→/LEAVE : 可以脱离自己参加的组队 
   ※不会询问 (Y/N)，所以要慎重考虑
#
→/EXPEL 【角色名称】: 组队长可将组队员，从队伍中删除
   ※不会询问 (Y/N)，所以要慎重考虑
#
→点选对话窗口右下段 [圆圈]后，将会显示队伍发言频道，点选后将可使用队伍频道与队员对谈
#
→%  : 在发言前加上%既可直接使用队伍频道与队员对谈，或是按住Ctrl键，再按Enter键，把输入的内容直接传达给队员  
#
→队员右上方微型地图将会显示各队员所在位置
#
→悄悄话 : 聊天窗口左边输入角色名称，右边输入对话内容 ，用TAB键将可快速于二个输入栏里转换位置
#
→选择微型地图 [+ / -] 符号时，地图将会 [放大 / 缩小] 
#
→[MVP系统] : 与Boss级魔物对决时，为了保护最高伤害者的权益，魔物的MVP物品将会直接掉落于MVP玩家，如魔物未掉落MVP物品，将补偿玩家50%的经验值 
#
→游戏在窗口模式进行时，可以移动窗口位置，按住[Alt] + [空格键]再按[M]键，可用^ff0000方向键或是鼠标^000000调整位置
#
→/fog : 可以把雾化效果开启/关闭，雾化效果是为各种地图做天气表现使用.
   对低阶计算机配备玩家来说，雾化效果会增加计算机负担，玩家如被迫跳出或停止游戏时，这时建议玩家关掉雾化效果，雾化效果也可组态设定下永久开启或是永久关闭
#
→从魔物掉下来的装备用物品、物品效果不能检定，可用道具商店所贩卖的放大镜来检定
#
→猎人若要用陷阱系列技能时，身上必须要有《携带装置用陷阱物品》
#
→组成工会需要消耗一个华丽金属，组成工会指令为  /GUILD  [工会名字] ,工会名字不可有特殊字符与空白，用Alt +G 可以打开工会窗口
#
→ 当工会解散时， " /breakguild[[工会名字] " 就可以解散工会，但工会员只剩下工会长1名时才可以使用
#
→工会会员之间聊天可使用 /gc 【对话内容】 或是 $ 【对话内容】 或是按 Alt + Enter 也可以
#
→想要组合的物品，放在组合窗口上点选卡片两次就可以组合，一定要把组合的物品，放在组合窗口上
#
→对 几种异常状态效果说明
  昏迷 : 变成昏迷状态，将不能行动
  黑暗 : 画面变黑暗，对魔物来说视线会变狭碍
  冰冻 : 将无法行动，当受到攻击时，才会解除此状态
#
→死亡时经验值会下降，使用技能后或是战斗状态后，需等待5秒才可下线
#
→使用仓库，纪录储存点，租用手推车，空间移动等，询问卡普拉职员即可
#
→[Alt] + [J] : 可以看到现在自己拥有的宠物情报
#
→角色追踪技能 : 按[Shift] + [鼠标右键]点选所想追踪的角色，便可以追踪角色
#
→[Alt] + [L] : 显示出现在可以使用的表情符号目录，可以把它移动到快捷工具栏上
